# TP2 Django - Gestion des Formations

Projet Django réalisé dans le cadre du TP2 à l’ENSAT.

## Description

Cette application permet de gérer :

- Les formations
- Les étudiants
- L’ajout, modification et suppression des données
- L’authentification avec une page de login

##  Technologies utilisées

- Python 3
- Django
- SQLite3
- HTML / CSS

##  Structure du projet

```bash
TP2_DJANGO/
│
├── env/
├── etudiants/
├── formations/
├── gestion_formations/
├── templates/
├── db.sqlite3
├── manage.py
└── README.md
````

##  Installation

### 1. Cloner le projet

```bash
git clone https://github.com/VOTRE_USERNAME/VOTRE_REPO.git
```

### 2. Accéder au dossier

```bash
cd TP2_DJANGO
```

### 3. Créer un environnement virtuel

```bash
python -m venv env
```

### 4. Activer l’environnement virtuel

#### PowerShell

```bash
.\env\Scripts\Activate.ps1
```

### 5. Installer Django

```bash
pip install django
```

### 6. Lancer le serveur

```bash
python manage.py runserver
```

##  Accès au projet

Ouvrir dans le navigateur :

```bash
http://127.0.0.1:8000/
```
